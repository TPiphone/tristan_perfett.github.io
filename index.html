The Data Engineering qualification from Stellenbosch University consists of a wide variety of courses and projects with the aim of preparing students for the Fourth Industrial Revolution. This includes all the tasks required to make data available for analysis, knowledge discovery and decision-making processes. The most important task of a data engineer is to develop and maintain an organization's data pipeline systems and implement algorithms to transform data into a usable format for analysis. The tasks of a data engineer include data collection, data storage, data synchronization, data transformation, data cleaning, data management, and data model development.
